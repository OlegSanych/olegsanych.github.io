Команда `find` в Linux используется для поиска файлов и каталогов на основе различных критериев, таких как имя, размер, дата модификации и т.д. Это мощный инструмент, который поддерживает множество опций.

## Общий синтаксис команды

```bash
find [путь] [условия] [действие]
```

- **путь** — каталог, в котором будет производиться поиск. Если не указан, поиск начинается с текущего каталога.
- **условия** — критерии, по которым производится поиск (например, имя файла, размер, дата).
- **действие** — что сделать с найденными файлами (например, вывести их на экран, удалить, скопировать и т.д.).

## Основные параметры и примеры

1. **Поиск по имени файла**

   Чтобы найти все файлы с определенным именем:

   ```bash
   find /путь/ -name "имя_файла"
   ```

   Пример:

   ```bash
   find /home/user/ -name "document.txt"
   ```

   Команда ищет файл с именем `document.txt` в каталоге `/home/user/` и всех его подкаталогах.

2. **Поиск без учета регистра**

   ```bash
   find /путь/ -iname "имя_файла"
   ```

   Пример:

   ```bash
   find /home/user/ -iname "document.txt"
   ```

   Команда будет искать файл `document.txt` или любой его вариант (например, `Document.txt`, `DOCUMENT.TXT` и т.д.).

3. **Поиск по шаблону**

   Для поиска файлов, удовлетворяющих определённому шаблону, можно использовать метасимволы:

   ```bash
   find /путь/ -name "*.txt"
   ```

   Пример:

   ```bash
   find /home/user/ -name "*.log"
   ```

   Команда найдет все файлы с расширением `.log`.

4. **Поиск по типу (файлы или каталоги)**

   Для поиска только файлов:

   ```bash
   find /путь/ -type f
   ```

   Для поиска только каталогов:

   ```bash
   find /путь/ -type d
   ```

5. **Поиск по размеру файла**

   Найти файлы определенного размера можно с помощью опции `-size`. Например, для поиска файлов больше 100 мегабайт:

   ```bash
   find /путь/ -size +100M
   ```

   Пример:

   ```bash
   find /var/log/ -size +500M
   ```

   Команда ищет файлы, размер которых больше 500 мегабайт в каталоге `/var/log/`.

   Размеры можно указывать с использованием следующих суффиксов:
      - `b` — 512-байтовый блок (по умолчанию);
      - `c` — байт;
      - `k` — килобайт;
      - `M` — мегабайт;
      - `G` — гигабайт.

6. **Поиск по времени модификации**

   Для поиска файлов, измененных за последние n дней:

   ```bash
   find /путь/ -mtime -n
   ```

   Пример:

   ```bash
   find /home/user/ -mtime -7
   ```

   Команда найдет файлы, измененные за последние 7 дней.

   Аналогично можно искать файлы по времени последнего доступа (`-atime`) или времени изменения метаданных (`-ctime`).

7. **Поиск по владельцу**

   Чтобы найти файлы, принадлежащие определенному пользователю:

   ```bash
   find /путь/ -user имя_пользователя
   ```

   Пример:

   ```bash
   find /home/user/ -user john
   ```

8. **Поиск по правам доступа**

   Для поиска файлов с определенными правами доступа:

   ```bash
   find /путь/ -perm права
   ```

   Пример:

   ```bash
   find /home/user/ -perm 755
   ```

9. **Выполнение действия с найденными файлами**

   По умолчанию `find` выводит на экран список найденных файлов, но можно выполнить другие действия. Например, для удаления найденных файлов:

   ```bash
   find /путь/ -name "*.log" -exec rm {} \;
   ```

   Здесь `-exec` позволяет выполнить команду `rm` для каждого найденного файла. `{}` представляет собой найденный файл, а `\;` завершает команду.

   Пример для копирования всех найденных файлов в другую директорию:

   ```bash
   find /путь/ -name "*.txt" -exec cp {} /backup/ \;
   ```

10. **Поиск и удаление пустых файлов и каталогов**

      Найти и удалить все пустые файлы:

      ```bash
      find /путь/ -type f -empty -delete
      ```

      Найти и удалить пустые каталоги:

      ```bash
      find /путь/ -type d -empty -delete
      ```

11. **Логическое объединение условий**

      Условия можно комбинировать с использованием операторов:

      - `-and` — логическое "И" (по умолчанию);
      - `-or` — логическое "ИЛИ";
      - `!` — отрицание.

      Пример:

      ```bash
      find /путь/ \( -name "*.txt" -or -name "*.log" \)
      ```

      Команда найдет файлы с расширением `.txt` или `.log`.

## Дополнительные опции

- `-maxdepth N` — ограничить глубину поиска до N уровней подкаталогов:

  ```bash
  find /путь/ -maxdepth 2 -name "*.txt"
  ```

  Команда ищет файлы только на глубине до 2 подкаталогов.
  
- `-mindepth N` — начать поиск с уровня N:

  ```bash
  find /путь/ -mindepth 1 -name "*.txt"
  ```

  Поиск начинается с первого подкаталога, пропуская сам каталог.

## Примеры

Команда                                                                          | Значение
---------------------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------
`find`                                                                           | рекурсивно показать все, что в директории
`find .`                                                                         | см. выше
`find . -print`                                                                  | см. выше
`find /var/lock \| less`                                                         | отправить в less то, что отработала команда `find`. далее пользуемся команадми less для навигации по результатам
`find . -name "*.json"`                                                          | поиск файла(ов) по маске
`find . -maxdepth 1 -name "*.json"`                                              | управление грубиной поиска (вложенности директорий)
`find . - type d -name "first"`                                                  | поиск только директорий или `f - файлов`
`find . - type d -path ./second -prune -o -name "*.json" -print`                 | исключить из поиска директорию `second`. Через `-o` можно добавлять доп условия. директории можно исключать/искать по шаблону `-name`
`find . -iname "*/JsOn"`                                                         | поиск без учета регистра
`find . -type f -atime 0`                                                        | искать файл по дате последнего открытия за последние 24 часа. +1 = сдвиг на 24 часа назад. Поиск файлов по аттрибутам (Modify/Access/Change/Birth)
`find . -type f -anewer filename`                                                | Найти файлы доступ к которым осуществлялся позднее `filename`
`find . -type f -cnewer filename`                                                | Найти файлы которые были изменены позднее чем `filename`
`find . -type f -size 140k`                                                      | найти файл определенного размера
`find . -type f -size -150k -a -type f -size +130k`                              | найти файлы в диапазоне объема 130-150к
`find . -type f -exec du -sh {} \;`                                              | выполнить команду с найденными файлами. В данном случае вывести их размер по шаблону
`find . -type f -exec du -sh {} \; \| sort -h`                                   | тоже самое, только с сортировкой
`find . -type f -exec du -sh {} \; \| sort -h \| tail -3`                        | тоже самое, только показать последние 3 файла
`find . -type f -exec du -sh {} \; \| sort -h \| head -3`                        | тоже самое, только показать первые 3 файла
`find . -type f -exec cat {} \;`                                                 | Вывести содержимое всех файлов
`find . -type f -exec ls -la {} \; -exec cat {} \; -exec echo "------------" \;` | Тоже самое только с указанием принадлежности содержимого файлу и разделитель "------------"
`find . -type f -name "*.txt"`                                                   | Найти файлы с расширением `*.txt`
`find . -type f -name ".*"`                                                      | Показать все скрытые файлы в текущем каталоге
`find . -type f -empty"`                                                         | Показать все директории в текущем каталоге
`find . -perm /a=x"`                                                             | Показать все иполняемые файлы в текущем каталоге
